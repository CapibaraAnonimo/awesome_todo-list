Para la ejecución de la aplicación con postgreSQL hara falta la instalación del paquete para esta base de datos del ORM typeorm usando "npm install --save @nestjs/typeorm typeorm pg"

Para la validación de datos descargamos las siguientes dependencias "npm i --save class-validator class-transformer"

Para la documentación usaremos instalaremos compodoc "npm install -g @compodoc/compodoc"
